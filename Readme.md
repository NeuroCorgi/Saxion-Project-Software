# Saxion vault project

## Requirements
1. arduino-cli
2. make
3. python3

## Main program
`make`

## Run tests

### Configuration
Configuration in the yaml file
#### Example configuration
```yaml
debug: false
pinName:
    mode: OUTPUT
    pin: 10
```

### Execution
`make test-(testname)`
testname-name of the component to test

####
Works only on Linux/MacOs
